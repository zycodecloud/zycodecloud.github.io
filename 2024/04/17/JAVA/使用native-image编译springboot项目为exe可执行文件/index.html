使用native-image编译springboot项目为exe可执行文件

# 安装 GraalVM JDK

  1. 下载并安装 GraalVM JDK

  1. 设置环境变量 JAVA_HOME 为 GraalVM JDK 的安装路径
  2. 添加 GraalVM JDK 的 bin 目录到 PATH 环境变量
  3. 运行 java -version 命令，检查是否成功安装和配置 GraalVM JDK

# 安装 native-image

  1. 打开 cmd。输入 gu install native-image (需要科学上网，如果下载不了，建议下载 GraalVM JDK 下的 jar 包进行导入)
  2. 输入 gu list 查看是否安装成功

# 安装 Visual studio

  1. 下载并安装 Visual Studio

  2. 选择 Visual Studio Community 2022

  3. 下载完成后打开安装文件

  1. 工作负载，选择使用 C++ 桌面开发

  1. 单个组件，搜索 msvc，选择 2-8，搜索 win，选择 win10--11 SDK

  1. 语言包，选择英文

  1. 点击安装

# 打包 springboot 为.exe 文件

  1. 新建 spring boot 项目，导入 Graalvm Native

  1. 勾选配置文件中的 native ，执行 clean ， native:compile

# 成功导出 exe 文件

启动速度也是非常之快

# 报错情况

# 1. 返回值不为零

解决方案

  1. 在开始菜单，进入 x64 Native Tools Command Prompt for VS 2022
  2. 输入 native-image @target\tmp\native-image-2476901968031795160.args

# 2. 找不到启动类

解决方案

  1. 在 pom.xml 文件中配置

  1. 在开始菜单，进入 x64 Native Tools Command Prompt for VS 2022

  2. 输入 native-image @target\tmp\native-image-2476901968031795160.args com.chx.hengxinmain.HengxinMainApplication
